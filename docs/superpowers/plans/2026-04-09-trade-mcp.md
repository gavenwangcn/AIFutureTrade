# trade-mcp Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增 `trade-mcp`（Spring Boot MCP Server）并补齐 `backend`/`binance-service` 缺失的 MVC 三层 Controller，使 MCP tools 能通过下游 Controller 完成账号/交易/市场数据查询，且 `klines_with_indicators` 的指标计算与 `trade/market/market_data.py` 1:1 对齐。

**Architecture:** `trade-mcp` 作为编排层，对外暴露 MCP tools；对内通过 HTTP 调用 `backend`（账号/交易，必须 `modelId`）与 `binance-service`（市场数据，不带 `modelId`）的 Controller。K 线指标由 `trade-mcp` 在内存计算并封装返回。

**Tech Stack:** Java 17, Spring Boot 3.2.x, Spring AI MCP Server（WebMVC），RestClient/WebClient，JUnit5
