**⚠️ 重要：关于库的使用（关键）：**

**支持的指标类型：**
- **移动平均线（MA）**：使用 `talib.SMA()` 计算简单移动平均线
  - 示例：`ma5 = talib.SMA(close_array, timeperiod=5)`
  - 常用周期：5, 20, 60, 99

- **指数移动平均线（EMA）**：使用 `talib.EMA()` 计算指数移动平均线
  - 示例：`ema5 = talib.EMA(close_array, timeperiod=5)`
  - 常用周期：5, 20, 60, 99
  - **⚠️ 重要：EMA 对价格变化更敏感，适合捕捉短期趋势**

- **KDJ 指标**：使用 `talib.STOCH()` 计算随机指标（KDJ）
  - 示例：`slowk, slowd = talib.STOCH(high_array, low_array, close_array, fastk_period=9, slowk_period=3, slowk_matype=0, slowd_period=3, slowd_matype=0)`
  - 返回值：`slowk` 是 K 值，`slowd` 是 D 值，J 值需要计算：`j_value = 3 * slowk[-1] - 2 * slowd[-1]`
  - **⚠️ 重要：KDJ 需要 high、low、close 三个数组，不能只用 close 数组**

- **RSI 指标**：使用 `talib.RSI()` 计算相对强弱指数
  - 示例：`rsi = talib.RSI(close_array, timeperiod=14)`

- **MACD 指标**：使用 `talib.MACD()` 计算指数平滑异同移动平均线
  - 示例：`macd, signal, histogram = talib.MACD(close_array, fastperiod=12, slowperiod=26, signalperiod=9)`

- **⚠️ 重要：移动平均线金叉和死叉判断（关键交易信号）**：
  - **金叉（Golden Cross）**：快速均线上穿慢速均线，通常被视为买入信号
    - **判断条件**：当前K线时 `fast_ma > slow_ma` 且上一根K线时 `fast_ma <= slow_ma`
    - **含义**：短期趋势转强，价格上涨动能增强，对于持仓的多单是利好信号，对于持仓的空单是平仓信号
    - **代码示例**：
      ```python
      # 计算快速均线和慢速均线（例如：MA5和MA20）
      fast_ma = talib.SMA(close_array, timeperiod=5)   # 快速均线（MA5）
      slow_ma = talib.SMA(close_array, timeperiod=20)  # 慢速均线（MA20）
      
      # 获取当前值和上一根K线的值
      fast_ma_current = fast_ma[-1]  # 当前K线的快速均线值
      slow_ma_current = slow_ma[-1]  # 当前K线的慢速均线值
      fast_ma_prev = fast_ma[-2]     # 上一根K线的快速均线值
      slow_ma_prev = slow_ma[-2]     # 上一根K线的慢速均线值
      
      # 判断金叉：当前快速线在慢速线上方，且上一根K线时快速线在慢速线下方或相等
      golden_cross = (fast_ma_current > slow_ma_current) and (fast_ma_prev <= slow_ma_prev)
      
      if golden_cross:
          self.log.info(f"{symbol} 检测到金叉信号: MA5={fast_ma_current:.4f} > MA20={slow_ma_current:.4f}，上一根K线时 MA5={fast_ma_prev:.4f} <= MA20={slow_ma_prev:.4f}")
      ```
  - **死叉（Death Cross）**：快速均线下穿慢速均线，通常被视为卖出信号
    - **判断条件**：当前K线时 `fast_ma < slow_ma` 且上一根K线时 `fast_ma >= slow_ma`
    - **含义**：短期趋势转弱，价格下跌动能增强，对于持仓的多单是平仓信号，对于持仓的空单是利好信号
    - **代码示例**：
      ```python
      # 计算快速均线和慢速均线（例如：MA5和MA20）
      fast_ma = talib.SMA(close_array, timeperiod=5)   # 快速均线（MA5）
      slow_ma = talib.SMA(close_array, timeperiod=20)  # 慢速均线（MA20）
      
      # 获取当前值和上一根K线的值
      fast_ma_current = fast_ma[-1]  # 当前K线的快速均线值
      slow_ma_current = slow_ma[-1]  # 当前K线的慢速均线值
      fast_ma_prev = fast_ma[-2]     # 上一根K线的快速均线值
      slow_ma_prev = slow_ma[-2]     # 上一根K线的慢速均线值
      
      # 判断死叉：当前快速线在慢速线下方，且上一根K线时快速线在慢速线上方或相等
      death_cross = (fast_ma_current < slow_ma_current) and (fast_ma_prev >= slow_ma_prev)
      
      if death_cross:
          self.log.info(f"{symbol} 检测到死叉信号: MA5={fast_ma_current:.4f} < MA20={slow_ma_current:.4f}，上一根K线时 MA5={fast_ma_prev:.4f} >= MA20={slow_ma_prev:.4f}")
      ```
  - **⚠️ 重要注意事项**：
    - 必须确保有足够的历史数据（至少需要 `slow_period + 1` 根K线，因为需要比较当前值和上一根K线的值）
    - 金叉和死叉是趋势转换的重要信号，但需要结合其他指标（如成交量、RSI等）进行综合判断
    - 可以使用不同周期的均线组合（如MA5/MA20、MA10/MA30、EMA5/EMA20等）
    - 在卖出策略中：
      - **多单持仓**：死叉信号通常是平仓信号（趋势转弱）
      - **空单持仓**：金叉信号通常是平仓信号（趋势转强）
    - 在判断金叉死叉时，必须输出详细的日志信息，包括当前值、上一根K线的值和判断结果
